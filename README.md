## 常见问题

**Q：搬瓦工官网打不开怎么访问？**

A：搬瓦工主域名在国内 DNS 被污染，可以通过官方镜像站访问，地址是 bwh81.net 或 bwh88.net，这两个是官方认可的镜像，功能完全一样。或者直接走本文的购买链接进入。

**Q：CN2 GIA-E 和 CN2 GIA 有什么区别？**

A：CN2 GIA-E（E 代表 Ecommerce）是电子商务系列的专属命名，线路本质上同样走 CN2 GIA，和其他 CN2 GIA 机房的区别主要在于带宽和机房位置。CN2 GIA-E 套餐最大的优势是可选机房多，一个套餐能切 12 个以上机房，灵活度很高。

**Q：哪个运营商用哪个机房更好？**

A：粗略说：电信用户选 DC6 CN2 GIA-E 或 DC9 CN2 GIA；联通用户选荷兰 EUNL_9 或者 DC6；移动用户选日本软银 JPOS_1 或者 CMIN2 线路机房。但最准的还是自己下单后每个机房都测一遍，CN2 GIA-E 套餐切机房免费，不用纠结。

**Q：搬瓦工支持 IPv6 吗？**

A：多数套餐带 /64 IPv6 段，具体看套餐说明页面。KiwiVM 面板里可以看到 IPv6 分配情况。

**Q：传家宝和限量版是什么，还能买到吗？**

A：传家宝是圈子里对 CN2 GIA-E 10G 限量版套餐的称呼，曾经年付 $49.99 就能买到 CN2 GIA-E 线路，性价比极高。限量版套餐库存极少，通常补货后几小时内售罄。目前常规在售的是上面表格里的那几档，限量版有没有货要靠运气，看官网库存页面是最直接的方法。
